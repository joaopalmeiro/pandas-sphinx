###############
 pandas-sphinx
###############

Render a `pandas <https://pandas.pydata.org/>`_ DataFrame as an
opinionated table for `Sphinx <https://www.sphinx-doc.org/>`_.

**************
 Installation
**************

Via `pip <https://github.com/pypa/pip>`_:

.. code:: console

   $ pip install pandas-sphinx
